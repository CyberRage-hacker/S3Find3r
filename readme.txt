# 🔍 S3 Bucket Finder by {CYBER RAGE}

A fast and flexible S3 bucket recon tool for bug bounty hunting and security research.  
It scans a list of target URLs and extracts potential Amazon S3 bucket names from their response content.

---

## ⚙️ Features

- Scans for S3 bucket patterns in website responses
- Supports multiple known S3 URL formats
- Randomized ASCII banners for cool CLI aesthetics 😎
- Fast, simple, and customizable
- Built with ❤️ by `{CYBER RAGE}`
